**Tweet 2**

Here's how it works:

1. Export your chat history from any platform
2. Drop it into the app — it auto-detects the format
3. A local AI embeds and indexes your messages
4. Pick a year and start chatting

Past-you responds grounded in your actual conversations.

**Tweet 6**

Tech stack for the curious:

- Tauri 2 (Rust + WebView)
- Gemma-3-4B via llama.cpp (GGUF)
- Nomic Embed for semantic retrieval
- MLX-LM for LoRA fine-tuning on Apple Silicon
- Zero npm dependencies in the UI
- ~2.5GB total model size
