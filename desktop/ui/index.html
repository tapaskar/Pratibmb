How to export your chats

WhatsApp

Open chatMenu ⋮MoreExport ChatWithout Media
  1. Open any chat in WhatsApp
  2. Tap Menu (three dots) or swipe down on the chat name
  3. Tap More → Export Chat
  4. Choose Without Media (faster, smaller)
  5. Save the .txt file to your computer
  6. Drag it into this window, or use the import button

Repeat for each chat you want to include. Group chats work too.

Facebook / Messenger

facebook.com/dyiRequest downloadJSONMessagesCreate File
  1. Go to facebook.com/dyi (Download Your Information)
  2. Click Request a download
  3. Select JSON format (not HTML)
  4. Under "Your information", select Messages and/or Posts
  5. Click Create File — Facebook will email you when it's ready (can take hours)
  6. Download and unzip the archive
  7. Point Pratibmb at the unzipped folder

This is the richest source for pre-2015 data. Posts, comments, Messenger DMs — everything.

  2. Choose JSON format
  3. Select Messages and/or Posts
  4. Request download — Instagram will email you
  5. Unzip and point Pratibmb at the folder

Gmail (Google Takeout)

takeout.google.comDeselect allMailNext stepCreate export
  1. Go to takeout.google.com
  2. Click Deselect all, then select only Mail
  3. Click Next step → Create export
  4. Wait for the download link (can take hours for large mailboxes)
  5. Download and unzip the archive
  6. Point Pratibmb at the .mbox file or the Takeout/Mail/ folder

Emails are threaded by References/In-Reply-To headers. Text-only (attachments are not imported).

  3. Point Pratibmb at ~/Library/Messages/chat.db

Quit Messages.app before importing. Both SMS and iMessage are included. macOS Ventura+ format supported.

Troubleshooting

No messages appeared after import

Check that you selected the right file. WhatsApp needs the .txt file, not the zip. Facebook and Instagram need the unzipped folder, not the zip archive. Also make sure the file is from a supported platform — Pratibmb auto-detects format but won't import unknown formats silently.

Messages are attributed to the wrong person

Your name in Pratibmb must match exactly how it appears in the export. Open the exported file and check what name your messages are tagged with. Update your name in Pratibmb to match (case-sensitive). In WhatsApp, this is your display name; in Facebook, your full name.

Characters look broken or garbled

The export file might not be UTF-8 encoded. This happens occasionally with older WhatsApp exports or emails. Try re-exporting from the platform. If you have the file, open it in a text editor and re-save as UTF-8.

WhatsApp: "Export Chat" option not visible

You must be inside an individual chat, not the chat list. Open the specific conversation first, then tap the three-dot menu. On iOS, swipe left on the chat name to see options. On some Android phones, it's under More → Export Chat.

Facebook: Export is HTML instead of JSON

Go back to facebook.com/dyi and make sure you select JSON format before clicking "Create File". HTML exports are not supported — they lose message structure that Pratibmb needs.

Facebook / Instagram export is taking forever

Large accounts can take 24-48 hours. Facebook and Instagram will email you when the download is ready. If it's been more than 48 hours, try requesting again with a smaller date range or fewer data categories (just Messages).

Gmail: MBOX file is too large

Go to takeout.google.com and use label filters. Instead of exporting all mail, select specific labels (Inbox, Sent, or custom labels) to reduce size. You can also limit the date range.

iMessage: Permission denied or no messages found

Three things to check: (1) Full Disk Access must be granted to your terminal app or to Pratibmb itself in System Settings → Privacy & Security → Full Disk Access. (2) Quit Messages.app before importing — it locks the database. (3) The path is ~/Library/Messages/chat.db (not iCloud).

Telegram: Export option is missing

The export feature is only available in Telegram Desktop (the full desktop app), not in the mobile app or Telegram Web. Download Telegram Desktop, then go to Settings → Advanced → Export Telegram data.

Twitter / X: Archive is not ready

Twitter archives can take 24-48 hours to prepare. You'll receive an email and a notification in the app when it's ready. If the link has expired, request a new archive from Settings → Your Account → Download an archive.

Discord: Do I need a bot token?

No. DiscordChatExporter uses your personal user token (found in browser DevTools), not a bot token. Follow the tool's documentation to extract your token. Only your own messages and channels you have access to will be exported.
